	\section{The Riscy Expedition - Prof. Arvind - 07 Jan. 2019}
	
	The talk was about leveraging open and free RISC-V ISA to support modular refinement, OOO (out-of-order) design and its impact on teaching. Speaker motivated about availability of large FPGAs in cloud and open source synthesis tools. Module composition using latency-insensitive (dataflow) framework and its limitations was discussed with an example. Hardware OO design using guarded atomic rules and cooperating Finite State Machines (FSM) was illustrated. The talk ended with the design of a complex superscalar processor design with non-blocking cache coherent memory system using conflict matrix and some case-studies from a related course.
